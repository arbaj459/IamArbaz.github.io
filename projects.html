1.Performance improvement in Graphics processing unit(GPU)

  • Description:

    In GPU, we have limited GPU memory size, and if we have applications that are larger than GPU memory, then even if we want to improve performance with GPU, we will not because our applications cannot fit into GPU memory. For this, virtual memory is used, but performance still does not get improve.To solve this problem, we have introduced a read-only duplication mechanism in the GPU.

  • Project :

    We have induced read-only duplication, in which whenever GPU memory is full, we will not evict read pages; instead, we will override read pages because of the eviction latency, which will save time and improve the performance of the GPU.
